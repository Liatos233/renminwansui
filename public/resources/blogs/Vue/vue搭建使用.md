# 1 API 风格

- 选项式 (Options API)

  - data、method、mounted 暴露在 this,指向当前组件实例

- 组合式(Composition API)

  - 与[`<script setup>`]配合使用

  - 导入的顶层变量 or 函数可以直接使用

> 选项式是在组合式的基础上实现的,组合式功能更加强大

# 2 Begin

1. 保证安装 nodejs>16.0

2. npm init vue@latest

3. cd projectName + npm install + npm run start

4. 使用 `Visual Studio Code` + `Volar 扩展`

5. npm run build
