#### 定义

- 一系列解决问题的指令
- 具有有限性、确切性、可行性

#### 复杂度

- 时间复杂度

  > Ο(1)＜ Ο(log n)＜ Ο(n)＜ Ο(nlog n)＜ Ο(n2)＜ Ο(n3)＜…＜ Ο(2^n)＜ Ο(n!)

- 空间复杂度
  - 单变量：O(1)
  - 一维数组： O(n)
  - 二维数组： O(n^2)
  - 动态分配、递归、栈

### 不使用临时变量交换变量值

- 解构

```js
[a, b] = [b, a];
```

- 加减法

```js
a = a + b;
b = a - b;
a = a - b;
```

- 异或

```js
a ^= b;
b ^= a;
a ^= b;
```
