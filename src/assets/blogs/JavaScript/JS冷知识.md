#### NaN

- 这个特殊的 Number 与所有其他值都不相等，包括它自己
- 唯一能判断 NaN 的方法是通过 isNaN()函数

```js
NaN === NaN; // false
isNaN(NaN); // true
```

#### 浮点数比较

- 要注意浮点数的相等比较
- 要比较两个浮点数是否相等，只能计算它们之差的绝对值，看是否小于某个阈值：

```js
1 / 3 === 1 - 2 / 3; // false
Math.abs(1 / 3 - (1 - 2 / 3)) < 0.0000001; // true
```

#### strict 模式

- 变量必须先声明后使用

- 禁止删除变量、函数或函数的参数

- 禁止使用保留字作为变量名（如 eval、arguments 等）

- 函数内部的 this 可能为 undefined，而不是默认绑定到全局对象。

- 禁止给只读属性赋值

- 函数的参数名称不能重复

- 禁止使用 with 语句

```js
"use strict";
```
